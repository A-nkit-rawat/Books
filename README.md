## Setup

### Backend

```bash
git clone https://github.com/A-nkit-rawat/Books/
cd Books/api
cp .env.example .env
DATABASE_URL=your_postgres_connection_string
GEMINI_API_KEY=your_gemini_api_key
npm install
npm run dev
npm install -g serve
serve Books/frontend
```
